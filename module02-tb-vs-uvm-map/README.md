# Module 02: Basic TB vs UVM map

**Kind:** `lab` · Primary lab: `tb-vs-uvm-map` · **Shipped**

[← TB layers](../module01-tb-layers/README.md) · [Course README](../README.md) · [TB anatomy refresh →](../module03-tb-anatomy/README.md)

## Outcomes

After this module you can explain and practice the ideas taught by **`tb-vs-uvm-map`**, as a browser sketch and/or offline UVM notes.

## Two tracks (pick one or both)

### Track A — Real UVM (hands-on)

1. Open [EXAMPLES.md](EXAMPLES.md) and read `examples/classic-vs-uvm/map.txt`.
2. Map classic habits to UVM roles in your own words.
3. Optional: skim course notes in [`../examples/verilator-uvm-hello/`](../examples/verilator-uvm-hello/).

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/tb-vs-uvm-map/index.html](http://127.0.0.1:8080/tools/tb-vs-uvm-map/index.html)
2. Live: [https://universal-verification-methodology.github.io/learning/tools/tb-vs-uvm-map/](https://universal-verification-methodology.github.io/learning/tools/tb-vs-uvm-map/)
3. Load the **starter example**, then work challenges.
4. Check off the Track B items in [CHECKLIST.md](CHECKLIST.md).

> Browser UVM tools are **concept literacy** — they do not replace Accellera UVM + a real simulator.
