# Module 07: Agent anatomy

**Kind:** `lab` · Primary lab: `uvm-agent` · **Shipped**

[← ConfigDB](../module06-uvm-configdb/README.md) · [Course README](../README.md) · [TLM ports →](../module08-uvm-tlm/README.md)

## Outcomes

After this module you can explain and practice the ideas taught by **`uvm-agent`**, as a browser sketch and/or offline UVM notes.

## Two tracks (pick one or both)

### Track A — Real UVM (hands-on)

1. Open [EXAMPLES.md](EXAMPLES.md) and read `examples/agent-sketch/anatomy.txt`.
2. Explain active vs passive and the roles of sequencer, driver, monitor.
3. Optional: grep `uvm_agent` in this module’s examples.

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/uvm-agent/index.html](http://127.0.0.1:8080/tools/uvm-agent/index.html)
2. Live: [https://universal-verification-methodology.github.io/learning/tools/uvm-agent/](https://universal-verification-methodology.github.io/learning/tools/uvm-agent/)
3. Load the **starter example**, then work challenges.
4. Check off the Track B items in [CHECKLIST.md](CHECKLIST.md).

> Browser UVM tools are **concept literacy** — they do not replace Accellera UVM + a real simulator.
