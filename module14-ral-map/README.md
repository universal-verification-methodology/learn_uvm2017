# Module 14: RAL map

**Kind:** `lab` · Primary lab: `ral-map` · **Shipped**

[← Scoreboard](../module13-uvm-scoreboard/README.md) · [Course README](../README.md) · [SVA timeline (lite) →](../module15-sva-timeline/README.md)

## Outcomes

After this module you can explain and practice the ideas taught by **`ral-map`**, as a browser sketch and/or offline UVM notes.

## Two tracks (pick one or both)

### Track A — Real UVM (hands-on)

1. Open [EXAMPLES.md](EXAMPLES.md) and read `examples/ral-sketch/map.txt`.
2. Explain block → register → field and front-door vs back-door access.
3. Optional: grep `uvm_reg_block` in the legacy UVM reg headers.

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/ral-map/index.html](http://127.0.0.1:8080/tools/ral-map/index.html)
2. Live: [https://universal-verification-methodology.github.io/learning/tools/ral-map/](https://universal-verification-methodology.github.io/learning/tools/ral-map/)
3. Load the **starter example**, then work challenges.
4. Check off the Track B items in [CHECKLIST.md](CHECKLIST.md).

> Browser UVM tools are **concept literacy** — they do not replace Accellera UVM + a real simulator.
