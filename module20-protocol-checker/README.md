# Module 20: Protocol checker

**Kind:** `lab` · Primary lab: `protocol-checker` · **Shipped**

[← Reporting](../module19-uvm-reporting/README.md) · [Course README](../README.md) · [VIP anatomy →](../module21-vip-anatomy/README.md)

## Outcomes

After this module you can explain and practice the ideas taught by **`protocol-checker`**, as a browser sketch and/or offline UVM notes.

## Two tracks (pick one or both)

### Track A — Real UVM (hands-on)

1. Open [EXAMPLES.md](EXAMPLES.md) and read `examples/checker-sketch/checker.txt`.
2. Explain checker vs scoreboard and three valid/ready rules.
3. Optional: grep `CheckerComp` in the legacy protocol_checkers example.

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/protocol-checker/index.html](http://127.0.0.1:8080/tools/protocol-checker/index.html)
2. Live: [https://universal-verification-methodology.github.io/learning/tools/protocol-checker/](https://universal-verification-methodology.github.io/learning/tools/protocol-checker/)
3. Load the **starter example**, then work challenges.
4. Check off the Track B items in [CHECKLIST.md](CHECKLIST.md).

> Browser UVM tools are **concept literacy** — they do not replace Accellera UVM + a real simulator.
